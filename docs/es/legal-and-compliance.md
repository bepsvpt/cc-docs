## Cumplimiento

### Cumplimiento de atención médica (BAA)

Si un cliente tiene un Acuerdo de Asociado de Negocios (BAA) con nosotros y desea utilizar Claude Code, el BAA se extenderá automáticamente para cubrir Claude Code si el cliente ha ejecutado un BAA y tiene activada la Retención Cero de Datos (ZDR). El BAA será aplicable al tráfico de API de ese cliente que fluye a través de Claude Code.
